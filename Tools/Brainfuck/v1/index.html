Symbolic Programming Language

What is Symbolic?

Symbolic is an esoteric programming language that uses only symbolic characters (no letters, numbers, or emojis). It operates on a stack-based system with a 2D memory grid and supports drawing, movement, input/output, and control flow.

Execution Model

• Stack: A last-in-first-out data structure for temporary values

• 2D Memory Grid: Infinite grid for placing blocks and data

• Cursor: Current position pointer starting at (0,0)

• Execution: Left-to-right, line-by-line processing

This example demonstrates stack manipulation and character output to print "Hello World!".

How to Program in Symbolic

Basic Concepts:

1. Stack Operations: Use + to push 1, - to pop, : to duplicate

2. Movement: Use >, <, ^, v to move the cursor

3. Drawing: Use # to draw a block at current cursor position

4. Loops: Use [ to start and ] to end loops (loops while top of stack is non-zero)

5. Output: Use @ for character output, $ for numeric output

Simple Examples:

+++++++++@    // Outputs character with ASCII value 9 (tab)
+++++[>+++++<-]>@    // Outputs character 'Y' (ASCII 89)
>#.>#.>#.    // Draw three blocks in a row
