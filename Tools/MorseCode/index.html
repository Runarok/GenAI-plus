About Morse Code Converter

Welcome!

This is a comprehensive Morse code converter that allows you to translate between regular text and Morse code instantly. The interface features a split-view design where both sides are fully editable and update each other in real-time.

How to Use

  • Type in either field: Enter text on the left or Morse code on the right
  • Live conversion: The other field updates automatically as you type
  • Copy buttons: Click the copy button in each section to copy content to clipboard
  • Clear all: Use the clear button to reset both fields at once
  • Customize: Change signal symbols in settings to match your preference

Supported Characters

Rules & Notes

  • Letters are separated by spaces in Morse code
  • Words are separated by " / " in Morse code
  • The converter is case-insensitive
  • Unsupported characters are ignored during conversion
  • Custom signal symbols can be changed in settings
